### Tiempos de Proceso Estimados
El proyecto tiene una duración total estimada de 152 días, distribuida entre varias etapas clave: el estudio técnico y las compra, la reorganización de la fábrica, la instalación de la celda robotizada y su programación y la integración del sistema MES junto con la capacitación:

|            Categoría             | Duración (Días) |
|:--------------------------------:|:---------------:|
|    Estudio   Técnico y Compras   |        43       |
|     Reorganización   Fábrica     |        91       |
|  Instalación   Celda Robotizada  |        23       |
|  Programación   Celda Robotizada |        18       |
| Integración   MES y Capacitación |        7        |
|          Total Proyecto          |       152       |
